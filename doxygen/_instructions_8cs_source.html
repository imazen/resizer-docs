19  /*
20  * A list of commands not implemented by this class, along with justifications
21  *
22  * maxwidth/maxheight - These will eventually be deprecated in favor of width=x;height=y;mode=max.
23  * Offering multiple variables for width and height leads to unexpected results, confusion, and consistency issues.
24  *
25  * color1,color2, angle - Justification - Commands are specific to one infrequently used plugin: Gradient
26  * speed - The SpeedOrQuality plugin doesn't have a great reason for existence, and is still alpha
27  * a.oilpainting, a.sobel, a.threshold, a.canny, a.equalize, a.posterize - Commands have neither been finalized nor confirmed.
28  * fi.scale - Specific to two very infrequently used plugins: FreeImageBuilder and FreeImageResizer
29  * preservePalette - Specific to the PrettyGifs plugin, and not well tested. *TODO*
30  * progressive - Specific to the FreeImage pipeline - not supported by GDI, WIC, or WPF
31  * shadow* - I'm hoping to retire/replace the DropShadow plugin sometime. I think it is very infrequently used.
32  * memcache - This feature is pre-alpha
33  * dpi - This feature is only useful if the user downloads the image before printing it. Lots of confusion around DPI, need to find a way to make it obvious. Perhaps naming it PrintDPI?
34  *
35  *
36  */
